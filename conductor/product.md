# rcn-web
Continuous web reconnaissance and vulnerability scanning automation.

# Product Guide - Alpha State
rcn-web is the web-reconnaissance implementation of the RCN ecosystem. It extends rcn-core with logic for asset discovery, traffic analysis, and vulnerability scanning.

## Target Audience
- Bug Bounty Hunters
- Penetration Testers / Red Teamers

## Key Capabilities
- **Automated Asset Discovery:** Finding subdomains, IPs, and ports.
- **Continuous Vulnerability Scanning:** Integration with tools like Nuclei.
- **Deep Traffic Analysis:** Utilizing Mitmproxy for flow inspection.
- **AI-Enhanced Context:** MCP integration for reconnaissance entries.
